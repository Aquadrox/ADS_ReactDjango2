Changer de branche : git checkout dev (via PyCharm ou terminal).

Coder : Modifier vos fichiers.

Stage : git add . (ou via l'interface PyCharm).

Committer : cz commit (dans le terminal PyCharm) et suivre l'assistant.

Pousser : git push origin dev (via PyCharm ou terminal).

(Plus2 tard) Merger : Fusionner dev dans master.

(Plus2 tard) Versionner : cz bump (pour taguer la nouvelle version).
